## Citation

## Simulator Design
The simulator uses discrete cycle time analysis to determine AGV fleet requirements.

## AGV Models
- XPL_201: Pallet truck (handover operations)
- XQE_122: Reach truck (rack storage, ground stacking)
- XNA_121/XNA_151: Narrow-aisle reach trucks (< 2.5 m aisles only)
